# argument

## 位置参数：

```python
>>> def foo(a, b):
...     print(a, b)
...
>>> foo(1, 2)
1 2
```

## 默认参数（如果值未传递，则使用默认值）：

```python
>>> def foo(a, b=3):
...     print(a, b)
...
>>> foo(3)
3 3
>>> foo(3, 4)
3 4
```

## 关键字参数

基于位置参数，区别在与调对参数显性赋值

```python
>>> def foo(a, b):
...     print(a, b)
...
>>> foo(a=1, b=2)
1 2
```

### 仅关键字参数

```python
>>> def foo(a, *b, c):
...     print(a, *b, c)
...
```
